Dependencies
------------
- Ruby 1.9.3
- [RVM](https://rvm.io)
- [Bundler](http://gembundler.com/)


Project Init
------------
    bundle install
    bundle exec middleman server

Project Build
-------------
Build using [Middleman Github Pages](https://github.com/newcontext/middleman-gh-pages)

    rake build
    rake publish
